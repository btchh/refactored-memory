Password Validation Test

This page tests the password strength and match validation functionality.

Test Cases:

  1. Weak password: Try "pass" - should show red bar and "Weak password"
  2. Medium password: Try "password123" - should show yellow bar and "Medium password"
  3. Strong password: Try "MyP@ssw0rd123!" - should show green bar and "Strong password"
  4. Password match: Enter same password in both fields - should show "✓ Passwords match" in green
  5. Password mismatch: Enter different passwords - should show "✗ Passwords do not match" in red
